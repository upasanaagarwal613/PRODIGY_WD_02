# ⏱ AI Smart Stopwatch Web Application

A modern and responsive Stopwatch Web Application built using **HTML, CSS, and JavaScript** with beautiful animated themes and productivity features.

---

## 🚀 Features

* ⏱ Stopwatch with Milliseconds Support
* 🎨 Animated Pink Floating Gradient Theme
* 🤖 Dark AI Cyber Theme
* ✍ Notes Feature (Saved with Timestamp)
* 💾 Notes stored using LocalStorage
* 🌙 Smooth Glassmorphism UI
* 📱 Fully Responsive Design

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Glassmorphism + Animated Gradients)
* JavaScript (Timer Logic + LocalStorage)

## 🎯 How It Works

1. Click **Start** to begin the stopwatch.
2. Click **Pause** to stop the timer temporarily.
3. Click **Reset** to clear the timer.
4. Add notes which automatically store the current stopwatch time.
5. Switch between themes using the Theme button.

## 💡 Key Concepts Used

* `setInterval()` for timer logic
* DOM Manipulation
* LocalStorage for persistent notes
